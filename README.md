# README #
The git repository for the timjedrek.com website.  Custom CMS made in Ruby in Rails.

Initialized on: August 14, 2022

Tech Stack:
- Rails
- Devise gem
- PostgreSQL
- Tailwind CSS
- Sendgrid
- AlpineJS
- Action Text / Trix Editor
- S3
- Dokku
- Linode
- Ubuntu

Version 1 deployed on: August 27, 2022


Future features:
